SalvaSenhas v2

SalvaSenhas version 2.0 (PasswordSaver, in a crude portuguese-english translation) is a software for storing passwords within a local SQLite database in the user's computer. The software uses double cryptography to safely store its passwords. Some of its features: Reports for each User, Suport for multiple users, Storage and Search of Passwords.
